# VendorBridge ERP

Procurement & Vendor Management ERP built using Next.js 14 App Router.

## Project Structure
- `app/` - App Router routes, page layouts, API endpoints, and server actions.
- `components/` - Shared UI and feature-specific client/server components.
- `lib/` - Application services, config, database client, and utility modules.
- `types/` - TypeScript domain entity models.

## Getting Started
1. Install dependencies: `npm install`
2. Setup environment variables: `cp .env.example .env`
3. Run the development server: `npm run dev`
